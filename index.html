About Me

I'm a huge video game fan, and spend the majority of my time either playing or making them. I also have an interest in Japan and have been studying Japanese for a few years, and enjoy translating things occasionally.
Here are some of the skills I've picked up doing the things I love:

Programming

I have been using C++, C# and Java for years, and have also used Python, Visual Basic, SQL to varying extents. I am currently learning DX11, OpenGL and more about C++ at University.

Version Control

I understand and know how to use the fundamentals of version control, and frequently make use GitHub and GitHub Desktop for my projects.

Creating Games

I have created games and game engines in C++, both 2D games with SDL and a 3D Physics engine with DX11.

I have been using the Unity Game Engine for more than 4 years, and have prototyped many different game types, ranging from 2D, 3D, Multiplayer and VR.

I helped create StaffsVerse, a virtual recreation of Staffordshire University in Unreal Engine 5 on a 6 week work experience scheme ran by the University.

Other Creations

I enjoy creating my own models and textures using Blender and image editing software such as Photoshop and Paint.Net. When it comes to simple SFX I simply adore BFXR.

I also have a small GitHub Repository of custom Blender Addons for when I decide to spend a few hours automating 10 minute tasks.
