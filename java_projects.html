Networked URealms Tool

This is a tool made to immitate one used in a D&D Roleplay show called 'URealms'. It is a simple Server & Client system in one jar, with images and sounds made by a friend. The Host has a control panel to control up to 4 sets of player data, and clients display this in a simple format.
It was made in 2018 and helped me learn simplistic networking, and was the first project I released.
